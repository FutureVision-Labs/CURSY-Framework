Quick Start

Get Started in 60 Seconds

Install CURSY Framework and start building beautiful synthwave interfaces.

Installation

1. Download the Framework

Copy the CSS files to your project or link directly:

<link rel="stylesheet" href="path/to/cursy.css" />

2. Include Google Fonts

<link href="https://fonts.googleapis.com/css2?family=Orbitron:wght@400;600;800&family=Inter:wght@300;400;600&display=swap" rel="stylesheet" />

3. Add Scanlines Effect (Optional)

<body class="scanlines">
